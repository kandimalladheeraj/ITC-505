Steps to Keep the Font

To keep the custom font displayed correctly, make sure to:

  1. Visit the Google Fonts website (fonts.google.com).
  2. Search for custom font as you like using the search bar.
  3. Click on the "Get Font" button which is located in top right
  4. From there you can download the font or you can embed code directly into your html page
  5. Copy the provided HTML link element and paste it into the section of your HTML document.
  6. Now you can use the selected font in your CSS by specifying font-family: 'karma', cursive;.
